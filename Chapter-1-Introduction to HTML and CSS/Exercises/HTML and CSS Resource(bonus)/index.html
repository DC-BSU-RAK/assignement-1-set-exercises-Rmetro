HTML & CSS Guide

HTML tags and CSS.

HTML Tags

1. Paragraph (<p>)

Defines a paragraph of text.

<p>This is a paragraph.</p>

2. Heading Tags (<h1> - <h6>)

headings, largest (<h1>) to smallest (<h6>).

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

3. Anchor Tag (<a>)

The code generates a functional link to another document page or specific area.

<a href="https://www.example.com">Visit Example</a>

4. Image Tag (<img>)

Displays an image on the webpage.

<img src="" alt="">

5. Ordered & Unordered Lists (<ol>, <ul>)

Defines ordered and unordered lists.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

CSS Properties

1. Color

Defines the text color.

p {
    color: blue;
}

2. Font Size

size of the text.

h1 {
    font-size: 24px;
}

3. Background Color

background color.

body {
    background-color: lightblue;
}

4. Margin

space around an element.

div {
    margin: 20px;
}
